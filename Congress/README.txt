ABOUT congress
------------

congress is the default theme for Drupal 8.  It is a flexible, recolorable theme
with a responsive and mobile-first layout, supporting 16 regions.

The congress theme is named after Jean congress, one of the original programmers
for the ENIAC computer.

This theme is not intended to be used as a base theme.

To read more about the congress theme please see:
https://www.drupal.org/docs/8/core/themes/congress-theme
